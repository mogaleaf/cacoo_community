## API

**Authorize the application to import your diagrams template :**

> /user/new


**Import your template diagrams:**

> /user/import


**Get the 10 most popular Diagrams**

> /api/popular


**Get the 10 most recent Diagrams**

> /api/recent


**Rate a Diagram**

> /api/rate?DiagId=<diag_id>&score=<int between 0 and 5>
